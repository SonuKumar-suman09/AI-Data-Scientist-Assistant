🤖 AI Data Scientist Assistant
An AI Data Scientist Assistant is a web-based application built with Python and Streamlit that makes data analysis and machine learning simple, interactive, and accessible. Whether you're a student, beginner, researcher, or data enthusiast, this application helps you perform the complete data science workflow without writing complex code.

Instead of using multiple tools for data cleaning, visualization, model building, and reporting, this application combines everything into one easy-to-use platform. Users can upload their own CSV or Excel datasets, explore the data through interactive dashboards, train machine learning models, compare their performance, and understand predictions using Explainable AI (SHAP).

🎯 Why This Project?
Many beginners struggle to perform data analysis because they need to learn multiple libraries and write hundreds of lines of code. This project simplifies the entire process by providing an interactive interface where users can analyze data, build machine learning models, and understand the results with just a few clicks.

The application not only predicts outcomes but also explains why those predictions were made using SHAP, making machine learning more transparent and trustworthy. This makes the platform useful for learning, academic projects, research, and real-world business analysis.

💻 Installation
Clone the repository:

git clone https://github.com/your-username/AI-Data-Scientist-Assistant.git
Move to the project folder:

cd AI-Data-Scientist-Assistant
Install the required libraries:

pip install -r requirements.txt
Run the application:

streamlit run app.py
